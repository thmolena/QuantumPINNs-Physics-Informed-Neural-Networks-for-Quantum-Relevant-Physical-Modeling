Research Project

QuantumPINNs

Physics-Informed Neural Networks for Quantum-Relevant Physical Modeling

This project is organized around one coherent question: when the governing quantum physics is known, how far can a PINN go without dense labeled solution data? The answer in this repository is built from three non-redundant studies: a specialist harmonic-oscillator benchmark, a specialist time-dependent Schrödinger benchmark, and a shared cross-problem benchmark that tests transferability rather than repeating the same accuracy claim.

The landing page uses only figures backed by CSV artifacts committed in the repository, so the benchmark figures shown here are stable and reproducible.

Research Logic

Three Studies With Different Jobs

The repository is strongest when read as a sequence. The harmonic-oscillator notebook establishes specialist stationary-state accuracy, the time-dependent notebook establishes structured complex-valued propagation, and the combined benchmark explains which design choices survive when the problem class changes.

Stationary-state benchmark

The harmonic-oscillator study jointly learns the wavefunction and eigenvalue. Its ground-state result is the most accurate numerical result in the repository.

Time-dependent benchmark

The Schrödinger study predicts the real and imaginary wavefunction components across space and time, then evaluates them with density and norm diagnostics.

Comparative benchmark

The combined study is intentionally broader than the specialist notebooks. Its role is to quantify transferability, architecture sensitivity, and robustness.

Reproducibility

Minimal Reproduction Path

The commands below match the environment assumption already used elsewhere in the repository.

conda activate qaoa
pip install -r requirements.txt

jupyter nbconvert --to notebook --execute --inplace notebooks/pinn_harmonic_oscillator.ipynb
jupyter nbconvert --to notebook --execute --inplace notebooks/pinn_schrodinger.ipynb
jupyter nbconvert --to notebook --execute --inplace notebooks/quantum_pinn_combined.ipynb

jupyter nbconvert --to html notebooks/pinn_harmonic_oscillator.ipynb --output pinn_harmonic_oscillator.html
jupyter nbconvert --to html notebooks/pinn_schrodinger.ipynb --output pinn_schrodinger.html
jupyter nbconvert --to html notebooks/quantum_pinn_combined.ipynb --output quantum_pinn_combined.html
python -m src.train --problem harmonic_oscillator --epochs 5000 --collocation 2000 --model-path model.pt
python -m src.server --model-path model.pt --problem harmonic_oscillator
python -m http.server 8000
