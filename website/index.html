PINN Inference Interface

Submit spatial and temporal coordinates to evaluate a trained PINN model via the local Flask API.

Input

Enter comma-separated values for x and optionally t.

Output

Run a prediction to see results here.

Quick Start

# 0. Use the project environment
conda activate qaoa

# 1. Train a model
python -m src.train --problem harmonic_oscillator --epochs 5000

# 2. Start the API
python -m src.server --model-path model.pt

# 3. Open this page
python -m http.server 8000
# → http://localhost:8000/website/
