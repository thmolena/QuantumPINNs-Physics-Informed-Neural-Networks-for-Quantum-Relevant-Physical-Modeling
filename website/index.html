Developer Report Interface

Submit spatial and temporal coordinates to inspect model inference through the local API.

This page applies the same report-oriented presentation as the project landing page while remaining focused on local evaluation. It is intended for controlled inspection of trained model outputs served by the Flask API in src/server.py.

Use Case
Local inspection of trained PINN predictions
Expected API
http://localhost:5000
Expected Payload
Comma-separated x values and optional t values

Inference Request

Enter comma-separated coordinates. When no time values are supplied, the interface uses t = 0.

Interface Notes

The API response is expected to include real and imaginary wavefunction components together with probability density.

The output panel preserves one line per queried coordinate pair to support quick inspection and manual spot checks.

This page is intended for local developer use and does not simulate results when the API is unavailable.

Inference Output

Prediction results are rendered exactly as returned by the local service after numeric formatting for readability.

Run a prediction to see results here.

Quick Start

# 0. Use the project environment
conda activate qaoa

# 1. Train a model
python -m src.train --problem harmonic_oscillator --epochs 5000

# 2. Start the API
python -m src.server --model-path model.pt

# 3. Open the interface
python -m http.server 8000
# http://localhost:8000/website/
